# ryanpdlittle.com

Hugo static site, custom theme at `themes/rpdl/`, deployed via GitHub Actions to GitHub Pages.

## Writing Style

When writing or editing blog posts, **always read the style guide first**: `~/Desktop/Projects/knowledge-hub/projects/ryanpdlittle-com/writing-style.md`

The style guide is the hub. It links to deep-dive docs on voice, narrative structure, prose craft, anti-AI rules, and media/code presentation. Read the hub every time, then load whichever deep-dive docs are relevant to the current task.

## Structure

- Posts: `content/posts/`
- Theme: `themes/rpdl/`
- Static assets: `static/images/`
- Deploy: push to master triggers GitHub Actions build
